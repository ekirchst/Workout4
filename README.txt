weatherbitAPI access program

*DESCRIPTION*
This is a simple program to access the weatherbitAPI. The program will output a json file containing all the data retrieved from the api, as well as generating a pdf of the daily weather stats.


*USE*
To use this program, change the "city" vaiable in the WorkoutProject4.py module to any city. *** IMPORTANT, Do not entera city with a space in it. For a city like New York, enter it as "NewYork". ***
